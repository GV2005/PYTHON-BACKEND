# Ecommerce API Testing Project

Built using Postman.

## APIs Tested

* Login API
* User Profile API
* Product Listing API
* Add to Cart API
* Cart Details API

## Features Implemented

* Authentication with Bearer Token
* Token extraction and storage
* Environment variables
* Collection variables
* Dynamic product selection
* Automated API tests
* Request chaining workflow

## Skills Demonstrated

* REST API testing
* POST / GET requests
* JSON request body
* Headers
* Authorization
* Response validation
* Test scripting in Postman
